Building Natural Chords

The Skip-Pick Method

Building Natural Chords from Your Scale

Here's the key insight: For chords that are natural to the major scale (like I, ii, iii, IV, V, vi, vii°), you use the SAME simple method - just stack every other note from your scale!

The Skip-Pick Formula:

Pick a note → Skip one → Pick a note → Skip one → Pick a note

The chord quality (major or minor) happens automatically based on which scale degree you start from!

💡 Note: This skip-pick method works for all the natural chords in the key. If you want a chord that's NOT natural to the scale (like VI major instead of vi minor), those are called "borrowed chords" - see the Borrowed Chords wiki page. The Calculate Chords tool handles them automatically!

Building Major Chords (I, IV, V)

Example: C Major (I)
C major scale: C - D - E - F - G - A - B

Start on C (1st):
1. C (start)
2. Skip D
3. E (pick)
4. Skip F
5. G (pick)
→ C + E + G
Example: F Major (IV)
C major scale: C - D - E - F - G - A - B

Start on F (4th):
1. F (start)
2. Skip G
3. A (pick)
4. Skip B
5. C (pick - wrap around to start of scale)
→ F + A + C

Building Minor Chords (ii, iii, vi) - Same Method!

You do exactly the same thing! Start on a different scale degree and skip-pick-skip-pick.

Example: Am (vi)
C major scale: C - D - E - F - G - A - B

Start on A (6th):
1. A (start)
2. Skip B
3. C (pick - wrap around)
4. Skip D
5. E (pick)
→ A + C + E

This automatically makes a minor chord because of the spacing between A-C-E!
Example: Dm (ii)
C major scale: C - D - E - F - G - A - B

Start on D (2nd):
1. D (start)
2. Skip E
3. F (pick)
4. Skip G
5. A (pick)
→ D + F + A

This is automatically minor!
💡 The Secret: You don't need different formulas for major vs minor! Just:
1. Use your major scale
2. Start on the right note
3. Skip-pick-skip-pick
The chord quality happens naturally!

Why Does This Work?

The skip-pick method works because it creates the intervals that define chord quality:

  • Major Chords: Have a bigger gap between the 1st and 2nd notes (major third)
  • Minor Chords: Have a smaller gap between the 1st and 2nd notes (minor third)
  • Diminished: Has small gaps between all notes (happens on vii°)

By using the skip-pick method from your scale, these intervals happen automatically!

🎹 Practice Tip: Pick a scale (like C major) and practice building all 7 chords using skip-pick:
I (C), ii (Dm), iii (Em), IV (F), V (G), vi (Am), vii° (Bdim)
Notice how you use the exact same method for all of them!
