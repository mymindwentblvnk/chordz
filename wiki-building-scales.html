Building Major Scales

Using the W-W-H-W-W-W-H Pattern

Using the W-W-H-W-W-W-H Pattern

Now let's use THE PATTERN to build a major scale from any root note:

W - W - H - W - W - W - H

Start on any note and follow this pattern to get your major scale!

Example: C Major Scale

Starting from C and applying W-W-H-W-W-W-H:

  1. Start: C
  2. W (Whole): C → D (skip C#)
  3. W (Whole): D → E (skip D#)
  4. H (Half): E → F (no key to skip)
  5. W (Whole): F → G (skip F#)
  6. W (Whole): G → A (skip G#)
  7. W (Whole): A → B (skip A#)
  8. H (Half): B → C (no key to skip)
C Major Scale: C - D - E - F - G - A - B - C
✓ All white keys! Perfect for beginners!

Example: G Major Scale

Starting from G and applying the SAME pattern W-W-H-W-W-W-H:

  1. Start: G
  2. W: G → A
  3. W: A → B
  4. H: B → C
  5. W: C → D
  6. W: D → E
  7. W: E → F# (need the black key to maintain the pattern!)
  8. H: F# → G
G Major Scale: G - A - B - C - D - E - F# - G
Note: F# is needed because the pattern requires a whole step from E
💡 Master This: The W-W-H-W-W-W-H pattern works from ANY starting note. Try it from D, F, A, or any key on the keyboard!

Why This Pattern Works

The W-W-H-W-W-W-H pattern creates the major scale's characteristic sound:

  • Bright and Happy: The specific intervals create a positive emotional quality
  • Stable Foundation: All major chords in a key are built from this scale
  • Universal: Every major scale in every key uses this exact pattern
