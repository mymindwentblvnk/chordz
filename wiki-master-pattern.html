The Master Pattern

W-W-H-W-W-W-H: The Foundation of Major Scales

🎹 THE MASTER PATTERN

This is all you need to memorize to play in ANY key:

W - W - H - W - W - W - H
Whole - Whole - Half - Whole - Whole - Whole - Half
💡 This pattern builds a major scale from ANY starting note!
Once you know this pattern, you can play in any key. Everything else in music theory builds on this foundation.

What Do Whole and Half Mean?

Quick Definition:
• Whole step (W) = Skip one key on the keyboard (e.g., C → D skips C#)
• Half step (H) = Move to the very next key (e.g., E → F, no key between)

Why This Pattern Matters

This pattern is the secret to understanding ALL major scales:

  • Universal: Works for any starting note - C, G, F#, anything!
  • Predictable: Once you know it, you can figure out any major scale without memorizing all 12 keys
  • Foundation: All chord progressions come from scales built with this pattern
  • Transferable: Learn it once, use it forever across all instruments

Example: Building the C Major Scale

Starting from C:

  1. Start: C
  2. W (Whole): C → D (skip C#)
  3. W (Whole): D → E (skip D#)
  4. H (Half): E → F (no key to skip)
  5. W (Whole): F → G (skip F#)
  6. W (Whole): G → A (skip G#)
  7. W (Whole): A → B (skip A#)
  8. H (Half): B → C (no key to skip)
Result: C - D - E - F - G - A - B - C
✓ All white keys! That's why C major is perfect for beginners.

Example: Building the G Major Scale

Starting from G:

  1. Start: G
  2. W: G → A
  3. W: A → B
  4. H: B → C
  5. W: C → D
  6. W: D → E
  7. W: E → F# (now we need a black key!)
  8. H: F# → G
Result: G - A - B - C - D - E - F# - G
Notice: Same pattern, different notes!
🎹 Practice Tip: Try building scales from different starting notes using this pattern. Start with C (all white keys), then try G (one black key), then D (two black keys). The pattern never changes!

Visual Pattern on Keyboard

On a piano keyboard, the pattern looks like this:

C C# D D# E F F# G G# A A# B

Notice that between E and F, and between B and C, there are no black keys. These are the natural half steps!
