How to Play Chord Progressions

From Roman Numerals to Keyboard Keys

🎹 THE MASTER PATTERN

This is all you need to memorize to play in ANY key:

W - W - H - W - W - W - H
Whole - Whole - Half - Whole - Whole - Whole - Half
💡 This pattern builds a major scale from ANY starting note!
Once you know this pattern, you can play in any key. Everything else on this page just explains how to use this pattern.
Quick Definition:
• Whole step (W) = Skip one key (e.g., C → D skips C#)
• Half step (H) = Next key (e.g., E → F, no key between)

1. Understanding Roman Numerals

Roman numerals represent chords built on different scale degrees. Here's what they mean:

Roman Numeral Chord Quality Example in C Major
I, IV, V Major chords (uppercase) C, F, G
ii, iii, vi Minor chords (lowercase) Dm, Em, Am
vii° Diminished chord Bdim
💡 Quick Tip: Uppercase = bright/happy major chord. Lowercase = darker/sadder minor chord.

2. The Chromatic Scale (All 12 Notes)

These are all the notes available on a keyboard, repeating every octave:

C C# D D# E F F# G G# A A# B

White boxes = white keys. Black boxes = black keys.

Note: C# and Db are the same key (enharmonic equivalents).

3. Using the W-W-H-W-W-W-H Pattern

Now let's use THE PATTERN to build a major scale from any root note:

W - W - H - W - W - W - H

Start on any note and follow this pattern to get your major scale!

Example: C Major Scale

Starting from C and applying W-W-H-W-W-W-H:

  1. Start: C
  2. W (Whole): C → D (skip C#)
  3. W (Whole): D → E (skip D#)
  4. H (Half): E → F (no key to skip)
  5. W (Whole): F → G (skip F#)
  6. W (Whole): G → A (skip G#)
  7. W (Whole): A → B (skip A#)
  8. H (Half): B → C (no key to skip)
C Major Scale: C - D - E - F - G - A - B - C
✓ All white keys! Perfect for beginners!

Example: G Major Scale

Starting from G and applying the SAME pattern W-W-H-W-W-W-H:

  1. Start: G
  2. W: G → A
  3. W: A → B
  4. H: B → C
  5. W: C → D
  6. W: D → E
  7. W: E → F# (need the black key to maintain the pattern!)
  8. H: F# → G
G Major Scale: G - A - B - C - D - E - F# - G
Note: F# is needed because the pattern requires a whole step from E
💡 Master This: The W-W-H-W-W-W-H pattern works from ANY starting note. Try it from D, F, A, or any key on the keyboard!

4. Building Natural Chords from Your Scale

Here's the key insight: For chords that are natural to the major scale (like I, ii, iii, IV, V, vi, vii°), you use the SAME simple method - just stack every other note from your scale!

The Skip-Pick Formula:

Pick a note → Skip one → Pick a note → Skip one → Pick a note

The chord quality (major or minor) happens automatically based on which scale degree you start from!

💡 Note: This skip-pick method works for all the natural chords in the key. If you want a chord that's NOT natural to the scale (like VI major instead of vi minor), you'll need to use interval formulas instead - but the interactive tool in Section 5 handles that for you!

Building Major Chords (I, IV, V)

Example: C Major (I)
C major scale: C - D - E - F - G - A - B

Start on C (1st):
1. C (start)
2. Skip D
3. E (pick)
4. Skip F
5. G (pick)
→ C + E + G
Example: F Major (IV)
C major scale: C - D - E - F - G - A - B

Start on F (4th):
1. F (start)
2. Skip G
3. A (pick)
4. Skip B
5. C (pick - wrap around to start of scale)
→ F + A + C

Building Minor Chords (ii, iii, vi) - Same Method!

You do exactly the same thing! Start on a different scale degree and skip-pick-skip-pick.

Example: Am (vi)
C major scale: C - D - E - F - G - A - B

Start on A (6th):
1. A (start)
2. Skip B
3. C (pick - wrap around)
4. Skip D
5. E (pick)
→ A + C + E

This automatically makes a minor chord because of the spacing between A-C-E!
Example: Dm (ii)
C major scale: C - D - E - F - G - A - B

Start on D (2nd):
1. D (start)
2. Skip E
3. F (pick)
4. Skip G
5. A (pick)
→ D + F + A

This is automatically minor!
💡 The Secret: You don't need different formulas for major vs minor! Just:
1. Use your major scale
2. Start on the right note
3. Skip-pick-skip-pick
The chord quality happens naturally!

7. What If The Progression Starts With Lowercase "i"?

Some progressions use a minor key (like i - VI - III - VII). Don't worry - it's easier than you think!

Two Simple Options:

Option 1: Use the Chord Generator (Easiest!)
The chord generator handles both major and minor keys automatically. Just select your root note and it shows you exactly what to play!
Option 2: Manual Method
If you see a progression starting with lowercase "i", you're in a minor key.

Minor scale pattern (different from major!):
W - H - W - W - H - W - W
Example: A minor scale
A → B (W) → C (H) → D (W) → E (W) → F (H) → G (W) → A (W)
Result: A - B - C - D - E - F - G - A
(All white keys - easiest minor key!)

Then use the same skip-pick-skip-pick method as before!
💡 Pro Tip: Most people find it easier to let the chord generator do this work! It automatically handles major/minor keys so you can focus on playing.
